## 💡 Customization

To customize this portfolio:

1. **Update Personal Info**: Edit the name, bio, and contact details in the HTML
2. **Modify Links**: Change social media URLs and contact links
3. **Adjust Colors**: Edit CSS variables in the `:root` section
4. **Add/Remove Skills**: Modify the skills-strip section
5. **Change Animations**: Adjust animation timing and effects in CSS
