# Naol Gonfa — Links

A personal link-in-bio page built with **React** + **Tailwind CSS**, deployed on [Render](https://render.com).

## Tech Stack

- React 18 + Vite
- Tailwind CSS v3
- Google Fonts (Space Grotesk, JetBrains Mono)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

Output goes to the `dist/` folder.

## Deploy on Render

1. Push this project to a GitHub repository.
2. Go to [render.com](https://render.com) → **New** → **Static Site**.
3. Fill in the fields as follows:

| Field | Value |
|---|---|
| **Name** | `naol-links` (or any name you like) |
| **Branch** | `main` |
| **Root Directory** | `NaolLink` *(if your repo root contains the NaolLink folder)* |
| **Build Command** | `npm install && npm run build` |
| **Publish Directory** | `dist` |

4. Click **Create Static Site** — Render will build and deploy automatically.
5. Every `git push` to `main` triggers a re-deploy.
